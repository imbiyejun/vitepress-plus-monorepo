# @imbiyejun/vitepress-plus-cli

Scaffolding tool for quickly creating VitePress Plus projects.

## Installation

```bash
npm install -g @imbiyejun/vitepress-plus-cli
```

## Usage

### Create New Project

```bash
vp-plus create <project-name>
```

### Initialize Existing Project

```bash
vp-plus init
```

## Commands

- `create` - Create a new VitePress Plus project
- `init` - Initialize VitePress Plus in existing project
- `add admin` - Add admin backend to project
- `upgrade` - Upgrade to latest version
