# About

This is an example documentation site built with **VitePress Plus**.

## What is VitePress Plus?

VitePress Plus is an enhanced version of VitePress that adds:

- Topic-based documentation organization
- Article status tracking (draft, planned, completed)
- Custom components for better content presentation
- Visual management backend (optional)

## Technology Stack

- VitePress 1.x
- Vue 3
- Ant Design Vue
- TypeScript
