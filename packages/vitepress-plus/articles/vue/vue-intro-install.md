## 什么是 Vue.js？

**Vue.js**（读音类似 view）是一个构建用户界面的渐进式框架。它被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。

### Vue.js 的特点

- **渐进式**：可以逐步采用，不必一次性重写整个项目
- **组件化**：将UI拆分成可复用的独立组件
- **响应式**：数据变化时，视图自动更新
- **声明式**：只需要描述状态与DOM的关系
- **易学易用**：学习曲线平缓，文档完善
